# Study Session Tracker

Alocal app I built to keep track of my study sessions.
I wanted something to log what I study, set targets, and see my weekly progress — so I made this.

## What it does
- Add subjects
- Set study targets (how many sessions and how long each session is)
- Edit them anytime
- Log study sessions during the week
- See progress per subject

## Note
Everything runs locally in the browser. Your data is saved on your device (no accounts, no server).

## Tech
- HTML
- CSS
- JavaScript
- LocalStorage

(Built with the help of AI tools)
